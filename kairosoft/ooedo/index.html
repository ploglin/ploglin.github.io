新手：這遊戲在玩什麼

你在江戶時代經營一座村莊，蓋城、鋪街道、開店招商，讓居民遷入並成長，一路把城鎮發展成全日本數一數二的大名。遊戲以「年／月」推進，通關後會解鎖快速模式。

衡量你身價的總指標是「石高（俸祿／Yield）」——所有建築、居民、名產都會貢獻石高，遊戲用石高全國排名與城鎮等級來評定你的地位。日文玩家間的經典挑戰，是「在 15 年內拿下石高全國第一」。

一句話目標提高石高＝提高排名。而拉高石高最有效的兩件事，就是築城與鋪小巷 combo。

石高與收入：城堡是關鍵

收入的核心是居民消費與「向城納品」。其中向城（Castle）納品是全遊戲最賺的收入來源，城還能為所有職業帶來收入，因此築城是最重要的成長事件。

  • 城要價 3000 兩，全遊戲最多可蓋 3 座，且無法拆除——位置要想清楚，最好蓋在未來的商業核心區。
  • 應提早存錢築城；太晚築城是常見的成長瓶頸。
  • 店舖的數值會隨顧客使用次數上升，商品售價跟著變高，所以要確保居民「走得到」你的店（留好道路）。

小巷 combo 怎麼觸發（3 格規則）

「小巷」（日文橫丁，官方繁中譯「小巷」，社群也稱街道組合）就是本作的 combo。規則和口袋學院系列不同，務必先搞懂：

  • 每個小巷需要一組指定建築（多為 3 種具體建築，是「固定配方」而非看類別）。
  • 觸發條件：這組建築彼此距離不超過 3 格——是「範圍 3 格內」，不必緊貼相鄰，比學院的擺放寬鬆。
  • 觸發後會提升該組及周圍鄰近建築的石高與商品價格兩項數值（分別標示，如 +25% 石高 / +10% 價格）。
  • 可疊加、可複用：不同小巷的效果會疊加，而且同一棟建築能同時參與多個小巷。這是佈局最大化的核心。
和口袋學院物語2 的差異學院是「把三種類別擠進一個 4×4 窗口」；大江戶是「指定的具體建築成組，成員彼此 3 格內」。所以大江戶更像「食譜」，一棟建築可跨多個食譜複用，佈局自由度更高。

高價值小巷推薦

若要優先鋪，先追高倍率的：

  • 開羅小巷（+35% / +35%）：全遊戲最強，但需開羅君雕像（4096 兩、最高聲望地標）等後期解鎖物，屬終盤收尾目標。
  • 關原小巷（+30% 石高）：砲台架＋砲台＋武器店，石高加成之王，軍事線解鎖後優先鋪。
  • 文明開化小巷（+25% / +25%）：電器店＋照相館＋動物園，石高與價格雙高，記得提早研究解鎖電器店。
  • 溫泉、火消、學者、開運、旅人、武士（+25% 級）：中後期主力，配方相對好湊。
效率心法把城與高階店集中在同一片區，讓多條小巷的加成在城堡周邊重疊，同一棟建築盡量參與多個小巷，收益最大化。

建築與住宅升級

住宅會依「租金比例」逐級升級，居民住得越好、貢獻越高：

  • 小屋（30%）→ 茅草屋（40%）→ 茅草房（需職等 3+，44%）→ 木造房（職等 4+，48%）→ 宅邸（需鄰近石板路，50%）→ 小城（65%，最高階住宅）。
  • 環境物件（樹木、花、大岩石、池、井等）擺在建築旁能提升鄰近石高，也是眾多小巷的必要成員，別只當裝飾。
  • 房子前鋪碎石路／草地、店前鋪石板路／碎石路，都能加石高並確保動線。

名產與研究

「名產」（Crafts）是中後期的放大器：

  • 取得：名產設計圖可在「批發商（問屋）附近的建築」發現；製作名產要消耗研究點數，由職人製作，居民的智力（Int）與技能（Skill）越高成功率越高。
  • 用途：把名產用在某居民或某建築上，可提升其石高或居民錢包；升級名產會增強效果並重新充能，可再次使用。
  • 研究分四類：食、軍事、技術、教育。研究點數來自居民購物、交談、製作名產與研究。
  • 名產與小巷都能吸引更多旅人造訪，是石高與收入的重要來源。

職業轉職與稅收

居民職業會依成長轉職：農民 → 商人 → 職人 → 武士 → 幕臣，要把各級練滿才能升上去。

  • 智力（Int）幫助發現新建築與升級；技能（Skill）幫助發現名產與取得研究點數——兩者都要顧。
  • 注意稅收規則：幕臣升到 5 級後就不再繳稅，規劃財政時要把這點算進去，別讓高階居民太多而稅收銳減。

佈局與衝分技巧

  • 2×2 為單位、外圈鋪路：建築以 2×2 擺放，外圈用道路包成 4×4，確保居民走得到，店家數值才會漲。
  • 用一棵「自然物」串多個小巷：很多 +10% 小巷第三格都收「自然物」，一棵樹可同時卡進好幾條小巷。
  • 集中高階店＋城：讓多條小巷加成在同一片區重疊，別把配方拆太散（記得成員要 3 格內）。
  • 提早存錢築城：3000 兩的城是收入引擎，越早蓋越滾得動；位置想清楚（不能拆）。
  • 提早研究終盤設施：電器店等要提早解鎖，才趕得上排名衝刺。
  • 常見錯誤：忘了留路導致店家沒客人、小巷成員超過 3 格沒觸發、太晚築城／太晚升幕臣。

常見問題

小巷一定要三個建築緊貼嗎？

不用緊貼，只要成員彼此在 3 格範圍內就會觸發，比口袋學院寬鬆很多。

一棟建築可以同時算進好幾個小巷嗎？

可以，而且加成會疊加。這正是佈局的關鍵——把常被引用的建築（自然物、和服店、圖書館、武家宅邸等）放在中心，一棟身兼多職。

城可以拆掉重蓋嗎？

不行，城無法拆除，且全遊戲最多 3 座，位置務必想清楚。
