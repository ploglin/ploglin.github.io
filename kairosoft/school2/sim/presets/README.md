## 多尺寸支援(已完成,2026-07)

- `gridRows`/`gridCols` 改為變數,`setGridSize(rows, cols, preserve)` 重建網格;工具列新增「地圖尺寸」下拉。
- `PRESETS` 每鎮帶 `rows`/`cols`;`data: null` 的城鎮可直接選取 → 以正確尺寸開空白畫布。
- 分享碼:非 26×24 時帶 `RxC;` 前綴;**26×24 維持無前綴,舊連結與外部產生器完全相容**。`decodeMap` 依前綴自帶尺寸解碼,`importGridData` 依陣列尺寸自動切換畫布(存檔/自動存檔/undo 快照同理)。
- 座標軸通用化:X = r+2、Y = gridCols+1−c(健康鎮不變;其他鎮的實際遊戲座標偏移待對照實機確認)。
- `scripts/check.js` 新增 27×27 前綴格式往返與「26×24 無前綴」守護檢查。
