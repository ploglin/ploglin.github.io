## 待辦:模擬器支援每張地圖不同尺寸

現行 `sim/index.html` 的 `gridRows=26`/`gridCols=24` 是寫死的。要支援其他四鎮需:

1. `PRESETS` 每個城鎮帶自己的 `rows`/`cols`,載入 preset 時重建網格。
2. 分享碼格式加入尺寸前綴(無前綴 = 26×24,**向下相容既有連結**);`decodeMap` 的 `cells.length !== gridRows*gridCols` 檢查改用該碼自帶的尺寸。
3. 座標軸標籤(X=r+2 / Y=25-c)可能因地圖而異,匯入實機地形時要對照遊戲內座標確認。
4. `scripts/check.js` 的 26×24 斷言改為依 preset 尺寸。
